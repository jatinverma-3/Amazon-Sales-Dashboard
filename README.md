# 📊 Sales Data Analysis Dashboard (Power BI)

## 🔍 Overview

This project focuses on analyzing sales data from multiple platforms such as Amazon, Walmart, and iHerb using Power BI. The goal is to derive meaningful insights and build an interactive dashboard for business decision-making.

## 📁 Dataset

The dataset includes:

* Product Category
* Revenue
* Units Sold
* Price
* Return Percentage
* Location
* Platform

## 📊 Dashboard Features

* KPI Cards (Total Revenue, Avg Price, Units Sold, Return %)
* Category-wise Revenue Analysis
* Platform-wise Revenue Distribution
* Location-based Sales Insights (Map)
* Product-level analysis
* Interactive filters (Category, Location, Platform, Date)

## 🛠 Tools & Technologies

* Power BI
* Excel
* DAX (Data Analysis Expressions)

## 💡 Key Insights

* Identified top-performing product categories
* Analyzed revenue contribution by different platforms
* Observed geographic trends in sales performance
* Evaluated return percentage for product efficiency

## 🚀 Project Outcome

This dashboard enables users to:

* Monitor sales performance
* Compare platform performance
* Make data-driven decisions
